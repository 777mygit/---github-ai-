## 项目背景

**项目**：飞书 × GitHub × AI 团队项目管理自动化工具
**仓库**：`https://github.com/777mygit/---github-ai-.git`
**核心文件**：`feishu/feishu_writer.py`（Markdown → 飞书块转换 + 推送）

工具把本地 `.md` 文件自动推送到飞书 Wiki，支持：
- Git commit 触发本地钩子自动推送
- GitHub Actions 云端推送
- 直接命令行推送（只推飞书，不走 Git）

## 今日解决的问题

### 问题 1：feishu_writer `--clear` 清空不彻底（已解决）
- **根因**：飞书 `batch_delete` API 单次最多删 50 个块，超出静默忽略
- **现象**：文档清空后追加，内容出现两遍（重复）
- **解法**：改为 `while True` 循环，每次删 50 个，直到 `list_children` 返回空
- **状态**：✅ 已修复合入，v1.4

### 问题 2：网络抖动导致推送中断（已解决）
- **根因**：飞书服务端关闭空闲 HTTPS 连接，脚本解析 Markdown 耗时触发
- **现象**：`ConnectionResetError (10054)` / `SSLEOFError`
- **解法**：`_call()` 加指数退避重试（3s/6s/9s/12s，最多 4 次）
- **状态**：✅ 已修复合入，v1.3
