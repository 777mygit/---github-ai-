# 附录 B：errno 常见错误码速查

| errno | 宏名 | 含义 | 常见场景 |
| --- | --- | --- | --- |
| 1 | `EPERM` | 操作不允许 | 非 root 操作需要特权的接口 |
| 2 | `ENOENT` | 文件或目录不存在 | open/stat 路径不对 |
| 4 | `EINTR` | 系统调用被信号打断 | read/write/accept 中途收到信号，需重试 |
| 5 | `EIO` | IO 错误 | 磁盘坏道、硬件故障 |
| 9 | `EBADF` | 无效的文件描述符 | fd 已关闭或从未打开 |
| 11 | `EAGAIN` | 资源暂时不可用 | 非阻塞 IO 无数据，select/epoll 就绪判断 |
| 11 | `EWOULDBLOCK` | 同 EAGAIN（Linux 上值相同） | 非阻塞 socket |
| 12 | `ENOMEM` | 内存不足 | malloc 失败、mmap 失败 |
| 13 | `EACCES` | 权限拒绝 | 文件权限不够 |
| 14 | `EFAULT` | 地址错误 | buf 是 NULL 或非法指针 |
| 16 | `EBUSY` | 设备忙 | trylock 失败、设备被占用 |
| 17 | `EEXIST` | 文件已存在 | O_CREAT\|O_EXCL 时目标已存在 |
| 19 | `ENODEV` | 设备不存在 | /dev 下设备未加载驱动 |
| 20 | `ENOTDIR` | 不是目录 | 路径中间段不是目录 |
| 21 | `EISDIR` | 是目录 | open 目录但没有 O_DIRECTORY |
| 22 | `EINVAL` | 无效参数 | 参数范围/类型错误，最常见的错误之一 |
| 23 | `ENFILE` | 系统 fd 表满 | 系统级 fd 耗尽 |
| 24 | `EMFILE` | 进程 fd 表满 | 超过 RLIMIT_NOFILE，需 setrlimit |
| 25 | `ENOTTY` | 非 tty | ioctl 发给非终端 fd |
| 28 | `ENOSPC` | 磁盘空间不足 | write/fsync 磁盘满 |
| 32 | `EPIPE` | 管道破裂 | 写入已关闭读端的管道/socket |
| 35 | `EDEADLK` | 死锁避免 | fcntl 记录锁检测到死锁 |
| 36 | `ENAMETOOLONG` | 文件名太长 | 路径超过 PATH_MAX(4096) |
| 38 | `ENOSYS` | 系统调用不存在 | 内核版本不支持该 syscall |
| 39 | `ENOTEMPTY` | 目录非空 | rmdir 非空目录 |
| 61 | `ENODATA` | 没有消息 | mq_receive 队列为空且非阻塞 |
| 98 | `EADDRINUSE` | 地址已占用 | bind 端口被占用，加 SO_REUSEADDR |
| 99 | `EADDRNOTAVAIL` | 地址不可用 | bind 了不属于本机的 IP |
| 104 | `ECONNRESET` | 连接被重置 | TCP 对端发了 RST |
| 107 | `ENOTCONN` | 未连接 | send/recv 前未 connect |
| 110 | `ETIMEDOUT` | 连接超时 | TCP 握手超时或 cond_timedwait 超时 |
| 111 | `ECONNREFUSED` | 连接被拒绝 | 对端端口没有监听 |
| 113 | `EHOSTUNREACH` | 主机不可达 | 路由问题，网络不通 |
| 115 | `EINPROGRESS` | 操作正在进行 | 非阻塞 connect 正在三次握手 |

> 使用 `strerror(errno)` 或 `perror("msg")` 打印人类可读的错误描述。线程安全版：`strerror_r(errno, buf, len)`。

# 附录 C：编译链接速查

| 需求 | 编译选项 |
| --- | --- |
| 开启调试信息 | `-g` 或 `-g3`（含宏展开信息） |
| 关优化（调试用） | `-O0` |
| 开全警告 | `-Wall -Wextra -Wpedantic` |
| 警告当错误 | `-Werror` |
| 链接 POSIX 线程 | `-pthread`（比 `-lpthread` 更全） |
| 链接实时扩展（mq/shm） | `-lrt` |
| 链接 libutil（pty） | `-lutil` |
| 链接数学库 | `-lm` |
| 链接动态库搜索路径 | `-Wl,-rpath,/path/to/lib` |
| 生成动态库 | `-shared -fPIC` |
| 生成静态库 | `ar rcs libfoo.a *.o` |
| 交叉编译 ARM | `arm-linux-gnueabihf-gcc` |
| 开 AddressSanitizer | `-fsanitize=address -g` |
| 开 ThreadSanitizer | `-fsanitize=thread -g` |
| 开 UBSanitizer | `-fsanitize=undefined -g` |
| 生成依赖文件 | `-MMD -MP`（配合 Makefile） |
| 位置无关可执行（PIE） | `-fPIE -pie`（现代发行版默认开） |
| 栈保护 | `-fstack-protector-strong` |
